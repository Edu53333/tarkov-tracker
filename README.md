# Tarkov Tracker v3

A high-performance, tactical progression tracker for Escape from Tarkov.

## Features
- **Real-time Data**: Powered by the tarkov.dev GraphQL API.
- **Deep Localization**: Full support for English and Spanish, including item and quest names.
- **Quest Tracking**: Filter by trader, map, and status. Hides locked quests automatically.
- **Hideout Manager**: View and track requirements for all hideout modules.
- **Item Logistics**: Unified view of all items needed for your current progression.
- **Persistence**: All progress is saved locally in your browser.

## Built With
- React + TypeScript
- Vite
- Tailwind CSS
- Lucide React

## Local Development
1. Clone the repository.
2. Install dependencies: `npm install`.
3. Start the dev server: `npm run dev`.
